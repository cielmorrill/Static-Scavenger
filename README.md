STARTING
- The game will boot up immediately upon running main.

MOVEMENT
- In order to move the robot, use the arrow keys.

SPRINTING
- In order to sprint, hold down the right or left shift key.
- Sprinting will decrease your sprint meter, which is colored blue.
- If your sprint meter is fully depleted, you will not be able to sprint unless it has been replenished.
- The sprint meter will refill over time as long as you are not trying to sprint.

ATTACKING
- Press the space bar to draw the robot's sword and attack.
- If the sword is not drawn, the robot will need to draw it first.
- If thee sword has not been used for a period of time, the robot will put it away.

COMBAT
- Getting hit by a slime will deplenish the robot's health, shown in yellow.
- If the robot's health fully depletes, it will die, and the game will end. It will need to be rebooted to continue.
- Slimes can be damaged and killed through the sword attack.

Developer's shortcut: Additional slimes can currently be spawned by clicking on the screen.
